# Athena-Xero (Athena Public Framework Pack)

This folder is meant to be dropped into an `athena-public` repo as a framework pack.

Contents:
- `.framework/`  : framework modules loaded on `/start` (Core Identity + Orientation)
- `.context/`    : source documents (Self-Structure parts, Principle 21, master outline)
- `.agent/`      : capabilities/workflows that can be referenced by the agent runtime
