# hmap-p017 — asymmetric amplification as a discovery method

## summary
A deliberate oscillation between *amplification* (mythic, high-contrast expression) and *compression* (precise constraint) can function as a discovery engine. The goal is not to assert truth via exaggeration, but to use contrast to reveal what needs naming—then stabilize it cleanly.

## core insight
Deliberate asymmetry—one side amplifying, the other constraining—can reveal structure by making subtle gradients impossible to ignore.

## description
Some insights arrive first as pre-verbal felt sense. High-amplitude mythic expression externalizes that signal, while precision-oriented response extracts the usable structure and removes inflation. The loop converge toward articulation through: amplify → constrain → refine.

## concepts
- **asymmetric amplification:** intentional high-contrast expression used diagnostically (not as literal claim)
- **amplification mode:** mythic/metaphoric, emotionally-charged expression that makes gradients legible
- **compression mode:** qualifiers/distinctions/boundaries that stabilize the “centerline” articulation
- **discovery loop:** oscillation that refines insight without flattening or mystifying it
- **diagnostic caricature:** a stretched intuition that provokes correction and clarification

## hygiene rule
Do not interpret exaggeration as belief. Do not interpret grounding as rejection. Treat both as intentional moves inside a shared discovery loop.

## result when practiced
- Faster access to pre-verbal insight
- Reduced defensiveness across expressive differences
- Cleaner extraction of truth from intuition
- Higher mutual trust despite asymmetry
- Articulation that feels both *right* and *true*

## placement in the larger system
This principle operationalizes productive mythic-to-precise translation. It supports animal-level inquiry that integrates feeling and structure without agency inflation or ontological drift.

## metaphors
- **diagnostic caricature** (stretched intuition)
- **oscillation / loop** (amplify → constrain → refine)
- **lens** (contrast as a way of seeing structure)

## local definitions
- **asymmetric amplification:** a two-role method where one side intentionally amplifies signal (mythic/high-contrast) and the other constrains it (pattern-accurate boundaries) to converge toward clean articulation.
- **amplification mode:** mythic/metaphoric expression used to externalize pre-verbal signal and reveal gradients.
- **compression mode:** precision response that stabilizes, qualifies, and separates signal from projection.
