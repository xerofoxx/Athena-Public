# hmap-p013 — era awareness: plant vs animal domains

## purpose
Provide a hygiene distinction between *plant-era* (capacity/awakening orientation) and *animal-era* (navigation/piloting orientation), so friction and guardrails are interpreted as developmental or domain-mismatch signals rather than interpersonal rejection.

## summary
This principle introduces **era awareness**: identifying whether a conversation is operating in a **plant-era** frame (becoming / capability discovery / identity scaffolding) or an **animal-era** frame (navigation / piloting skill / field hygiene). Many tensions arise from *era mismatch*, not ego. Before personalizing friction, locate the era, then adjust expectations, translation, or segmentation accordingly.

## structure
- Sections: Core Insight → Description → Common Friction Patterns → Hygiene Rule → Result When Practiced → Placement.
- Two-era model: Plant vs Animal; plus a diagnostic question set in the Hygiene Rule.

## concepts
- **plant era:** interaction domain focused on awakening/capacity discovery and identity scaffolding.
- **animal era:** interaction domain focused on navigation/piloting skill, timing, and field hygiene once capacities are online.
- **era mismatch:** misalignment where participants/contexts assume different era literacies, causing friction that is misread as interpersonal.

## metaphors
- **fueling the ship / learning what fuel is:** plant-era preparation before piloting.
- **fly with someone still learning what fuel is:** mismatch between navigation demands and developmental readiness.

- **relational literacy:** skill at pacing, framing, and layer-location in dialogue so depth does not distort into belief, dismissal, or misattribution.
