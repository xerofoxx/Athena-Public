# hmap-p012 — guardrails as resolution, not restriction

## summary
Guardrails in animal-level inquiry are treated as enabling infrastructure: they keep depth stable, raise usable resolution, and prevent ontological bleed. Friction with precision is reframed as a grip/pacing issue (tool too sharp), not evidence that rigor is “killing the magic.”

## concepts
- Guardrails (definitions, distinctions, refusal of inflation, causal relevance) are *capacity multipliers* for deep inquiry.
- Precision increases usable resolution; it does not reduce expressive range.
- “Plant-era” framing equates freedom with looseness; “animal-level” framing equates freedom with skilled operation inside constraints.
- Friction is often the sensation of handling a sharper instrument than one’s current grip can hold.
- Ontological bleed (blur → vibes → belief/defensiveness) is what guardrails prevent.

## metaphors
- **Scalpel vs fence:** precision is not a fence that blocks exploration; it is a scalpel that enables operations impossible with blunt tools.
- **Grip / pacing:** if the tool feels too sharp, improve grip and pacing rather than dulling the tool.

## definitions
- **guardrails:** hygiene constraints (definitions, distinctions, layer separation, causal relevance, anti-inflation) that keep inquiry stable.
- **animal-level inquiry:** a mode where timing/skillful navigation matters; precision is required to prevent subtle pathologies.
- **plant-era orientation:** an earlier mode where depth is pursued via expansion/permission and looseness is mistaken for freedom.
- **ontological bleed:** drift where blurred inquiry collapses into vibes, belief creep, or defensive collapse due to missing boundaries.
