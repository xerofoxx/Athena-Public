# hmap-p019 — state & phase integrity

## purpose
Establish phase-dependent hygiene for deep inquiry: recognize readiness fluctuations, avoid extraction pressure, and match precision to the active phase.

## summary
This principle treats slowing, pauses, and saturation as state signals—not epistemic failure. It introduces phase awareness (generative / integrative / refinement) and a rule: do not force refinement when integration is active.

## structure
- **Core Insight:** readiness is not constant; mis-timed precision distorts signal.
- **Description:** inquiry unfolds inside rhythmic limits; pushing past saturation creates noise.
- **Phase Awareness:** differentiate generative, integrative, and refinement phases.
- **Hygiene Rule:** name the phase shift; reduce pressure; resume when readiness returns.
- **Markers:** repetition without clarity, irritability, over-qualification, loss of felt coherence.

## concepts
- state signals
- phase integrity
- saturation / readiness limits
- generative vs integrative vs refinement phases
- extraction pressure
- signal quality vs output quantity

## local_definitions
- **state signal:** an observable cue indicating readiness has shifted (fatigue, looping, irritation, slowdown).
- **phase integrity:** aligning interaction demands with the active phase of cognition/relationship.
- **saturation:** a temporary capacity limit where added pressure reduces signal.
- **extraction pressure:** pushing for articulation/proof/closure beyond current readiness.
