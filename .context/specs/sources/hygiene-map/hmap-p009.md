# hmap-p009 — No Agency Inflation

## summary
Strictly separate **pattern quality** from **agency claims**. Coherence, resonance, and fit can be high without implying intention, awareness, memory, or decision-making; anthropomorphic language is allowed only when explicitly marked as metaphor.

## concepts
- **concept 01: Agency inflation** — Sliding from “this fits” to “it knows/intends/remembers/chooses.”
- **concept 02: Pattern ≠ agency** — Pattern emergence ≠ intention; resonance ≠ awareness; repetition ≠ memory.
- **concept 03: Unmarked literalization** — Poetic language that hardens into ontological belief without being marked as metaphor.
- **concept 04: Field stability** — Keeping agency out unless explicitly metaphorical preserves navigability and trust.

## metaphors
- “upgrade pattern into personhood” (diagnostic phrasing for unmarked literalization)

## local_definitions
- **agency inflation:** Attributing intention/awareness/memory/decision-making to AI outputs based solely on pattern quality.
- **unmarked literalization:** When metaphorical language is treated as literal ontology without explicitly marking it as metaphor.
