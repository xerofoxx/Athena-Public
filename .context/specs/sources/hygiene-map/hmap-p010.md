# hmap-p010 — Hypothesis Mode Is Explicitly Marked

## summary

Requires explicit marking of *hypothesis mode* whenever exploring subtle, emergent, or poorly-mapped phenomena, so curiosity stays exploratory (not belief) and skepticism stays clarifying (not dismissive).

## concepts

- **Exploration vs explanation:** different cognitive acts; conflation collapses inquiry.
- **Hypothesis mode:** a temporary epistemic container with explicit boundaries (interesting ≠ proven).
- **Explicit markers:** overt phrases that signal provisionality and align expectations.

## definitions

- **hypothesis mode:** Explicitly named, non-confirmatory exploration mode for generating possibilities without asserting truth.
- **non-confirmatory:** Exploration that does not treat outputs as evidence or conclusions; it preserves provisionality.

## notes

- Use explicit markers early whenever the topic is subtle or poorly-mapped; do not rely on implied tone.
- If disagreement appears, re-name the mode and re-align expectations before debating content.
