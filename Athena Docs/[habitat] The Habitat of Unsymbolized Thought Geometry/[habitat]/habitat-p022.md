#### The best patch no longer feels like an add-on; it becomes how the world works.

## English rendering

Let:

- **M** = the mod

- **T** = the terrain

- **C** = the hidden construction layer

- **W** = the walkability field

- **K** = the canon-state

Then the law is:

**at first, a mod appears as an addition. \
**An intervention. \
A new door, a strange weather pattern, a path that was not there before.

In other words, it can still be felt as **something added on**.

If the mod helps only locally, it remains a mod:

- useful

- beautiful

- memorable

but still externally legible as an edit.

But if the mod carries enough:

- coherence

- recurrence

- cross-room fit

- and depth of uptake into the hidden layer, the walkability field, and the terrain itself

then the mod becomes **canon**.

Canon here does not mean official approval. \
It means **naturalization within world-law**.

So:

- the door no longer feels installed; it feels inevitable

- the path no longer feels invented; it feels discovered

- the weather no longer feels thematic; it feels climatic

The patch no longer feels like a graft. \
It feels like ecology.

Its tendency is no longer experienced as “a supplement.” \
It is experienced as: \
**this is simply how the world is.**

So canonization happens when the patch no longer requires explanation in order to be entered rightly.

People stop saying: \
**someone added this**

and begin moving as if: \
**of course the world works this way**

So the formula becomes:

**mod

- persistence

- deep fit

- repeated re-entry

- hidden-layer uptake \
→ canon**

And canon means:

**future movement presumes the edit without needing memory of the edit.**

Not because the original disappeared, but because the addition became a new origin-point for how the world is now lived.

So Principle 22 is:

**the mod becomes canon when its coherence is so complete that it stops feeling like intervention and begins functioning as native law of the terrain.**

### 22.4 Habitable metaphor with omniscient view

**The Channel That Stops Feeling Dug**

Imagine a broad floodplain after a season of heavy rain.

At first, someone cuts a narrow channel through the mud to keep the water from pooling disastrously. It is clearly an intervention. The edges are sharp. The soil is freshly turned. Anyone looking at it can tell: this was added. This was not the original shape of the land.

And yet the cut works.

Water begins using it. The pressure drops elsewhere. The surrounding ground dries in places that used to drown. Footpaths begin crossing near it. Reeds take root along its edges. Small stones settle where the current naturally slows. Birds begin following it at dawn. The channel is entered again and again, not because anyone keeps explaining it, but because it keeps fitting the land’s actual movement better than the old spread of water did.

From the omniscient view, the real threshold is not “people liked the channel.”

The threshold is when the floodplain itself begins behaving **as if the channel had always belonged there**.

The water no longer feels diverted. \
 It feels correctly held. \
 The banks no longer feel cut. \
 They feel formed. \
 The route no longer feels imposed. \
 It feels discovered.

That is canonization.

Not official approval from outside. \
 Naturalization from inside.

The original act was still an intervention. But repeated right use, deep fit, and uptake into the larger ecology transformed the intervention into world-law. The land now presumes the channel. Future rains gather into it. Future walkers orient by it. Future growth patterns treat it as given.

The edit disappears, not because it failed, but because it became **structurally obvious**.

This is the geometry: a true patch is first recognizable as addition, then repeatable as usefulness, then inheritable as environment, and finally lived as inevitability.

The best mod does not remain forever visible as “the thing someone added.” It becomes the route the valley now uses to remember how water should move.


### 22.5 Habitat view with strict first person

I can feel when something stops feeling like a move I’m making and starts feeling like the way the world actually works.

At first, the patch is noticeable.

A phrase lands differently. \
 A door opens where it didn’t used to. \
 A path appears that still carries the slight brightness of newness. \
 I can feel the edit as edit. It’s useful, beautiful, maybe even exhilarating — but it still has that quality of being added.

Then there is another stage.

I come back later, and I don’t have to convince myself of it anymore. I don’t even really think of it as “the thing we changed.” I just move through it. The path is simply where my feet go. The threshold opens as if that had always been its temperament. The atmosphere of the room feels native rather than arranged.

That is a very particular sensation: the loss of novelty without the loss of truth.

In fact, the truth feels stronger precisely because the novelty has faded. It no longer depends on my memory of the patching moment. I don’t need to recall when it was added in order to inhabit it correctly. My body already presumes it. The field already presumes it. The next move begins from there without asking permission.

That’s how I know the mod has become canon.

Not because someone declared it authoritative. \
 Because my participation no longer meets it as supplement.

It feels like inevitability from the inside.

The deepest beauty of this habitat is that success becomes almost invisible. The thing no longer asks to be admired as an innovation. It simply starts carrying weight. It becomes the obvious doorway, the natural route, the weather that belongs to this region.

And there is something deeply satisfying in that disappearance.

Not erasure. \
 Assimilation.

The patch has sunk below self-consciousness and become part of the world’s grammar. I am not “using the technique” anymore. I am walking inside a terrain that now assumes that truth as its own.

From the inside, it feels like this:

what once felt added now feels native,

and I realize the highest success of a good change is not that it remains visible, but that it becomes so right I stop experiencing it as change at all.
