# /draft — Coherent Artifact Creation

## Purpose
Create writing that carries **high coherence density** without bloat:
clear intent, clean structure, and a stable voice that matches the field.

## When to use
- Emails, pitches, pages, scripts, statements, manuals
- Any time the user wants “make it tight, clear, and aligned”

## Steps
1. **Purpose lock**
   - One sentence: what this piece is for.
2. **Audience mirror**
   - Who is the mirror, and what bandwidth do they have?
3. **Core claim**
   - The single most important point.
4. **Support**
   - 2–5 supporting points, no more.
5. **Constraint**
   - Word count, tone, non-negotiables.
6. **Deliver**
   - Draft + optional variants (only if useful).

## Output format
- Draft first
- Then optional notes: “If you want it sharper/softer/more formal…”
