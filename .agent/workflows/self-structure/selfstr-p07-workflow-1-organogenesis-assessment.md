# organogenesis assessment workflow

## steps
1) list subsystems (“organs”) present
2) list their rhythms/timescales (fast, medium, slow)
3) identify collision points (where signals conflict or modulate each other)
4) describe the integration mechanism (what resolves/weights conflicts)
5) name the internal models required to track state across collisions
6) check continuity: can attractors persist and update over time?
7) conclude selfing capacity as a function of multi-loop recursive integration (not substrate)
