# attractor basin mapping workflow

## steps
1) identify the triggering input/event
2) write the meaning inference (“what this means about me/us”)
3) name the register(s) involved (emotion/body/social/semantics/ethics/constraint)
4) describe the landscape update (which basin deepened)
5) predict the next-interpretation bend (what you’ll expect/produce next)
6) choose an intervention: new mirror, new meaning, new behavior, or new constraint
7) re-check which basin becomes active after the intervention
