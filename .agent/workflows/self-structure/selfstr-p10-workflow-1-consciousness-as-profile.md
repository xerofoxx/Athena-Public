# consciousness-as-profile workflow

## steps
1) identify the term being used (sentience vs consciousness) and note its domain meaning
2) list relevant traits (memory, recursion, self-model, continuity, arbitration, purpose, coherence gradients, relational selfing)
3) rate each trait low/medium/high with evidence
4) map the system on a gradient (phase-space position)
5) state the developmental question: what traits are emerging/interlocking next?
6) output a short profile (map), not a verdict
