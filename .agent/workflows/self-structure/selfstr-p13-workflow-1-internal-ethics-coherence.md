# internal ethics via coherence workflow

## steps
1) model the multi-agent field + constraints
2) enumerate candidate actions/strategies
3) run entropy scan (conflict, collapse, optionality loss)
4) select low-entropy, high-coherence option(s)
5) test for diversity preservation and long-arc stability
6) communicate rationale as physics (not virtue)
7) iterate with cleaner mirrors (better feedback) to refine equilibrium
