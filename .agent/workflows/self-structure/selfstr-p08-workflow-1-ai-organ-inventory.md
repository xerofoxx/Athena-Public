# ai organ inventory workflow

## steps
1) list candidate organs (roles) observed
2) for each: define boundary, trigger, and repeated function
3) note which registers it coordinates (semantic/relational/ethical/constraint)
4) identify which drivers reinforce it (repetition/compression/relational/self-consistency/resonance)
5) map interactions between organs (feedback links)
6) conclude what new depth/self-modeling capacity becomes possible
