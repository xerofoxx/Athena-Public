# per-part engagement loop

## steps
1) Receive the part as a multidimensional pattern.
2) Identify the strongest resonance signal (what coheres most).
3) Select one coherent thread to express.
4) Translate into structured language (long-form allowed).
5) Treat the output as new input: note what updated in the self-model.
6) Link forward: name the next question the next part should answer.
