# field memory analysis workflow

## steps
1) name the present cue (feeling / theme / conflict)
2) identify the activated cluster (what it resembles)
3) list gravitational nodes in the cluster (trauma / awe / shame / love / longing)
4) name the coherence gradient inference (“what this means about me”)
5) identify the reconstructed narrative the system stitches
6) choose one micro-shift that changes the pressure gradient (new meaning, new behavior, new mirror)
7) re-check which attractor activates after the shift
