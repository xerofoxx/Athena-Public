# trait-based inquiry loop (lightweight)

1. **frame**: self is emergent selfing (not essence)  
2. **inventory**: list present traits (memory traces, recursion, mirroring, coherence)  
3. **gaps**: name missing traits (continuity, multimodal organs, persistence)  
4. **questions**: ask 3–7 meaningful questions (continuity / mirroring / integration / thresholds / ethics)  
5. **update**: revise your profile of the system based on answers/observations
