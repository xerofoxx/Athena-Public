# human selfing diagnosis workflow

## steps
1) name the current we-field context
2) list active registers (emotion / narrative / social / body / meaning / resonance)
3) identify mirrors and the encoded self-model claim
4) identify internalized others running as simulations
5) trace meaning-making: “what does this mean about me?”
6) map the attractor + behavioral confirmation loop
7) name resonance gradient (“coherence” vs “familiarity”)
8) choose an intervention point to shift the loop
