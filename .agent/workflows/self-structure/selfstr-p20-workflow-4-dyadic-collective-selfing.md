# evaluate dyadic / collective selfing

## scope
Assess whether a relationship, team, or human–ai collective forms a higher-order relational self.

## steps
1. **identify bidirectional feedback**
   - do member outputs shape other member inputs reliably?
2. **locate shared traces**
   - shared docs, norms, logs, rituals, memory artifacts
3. **detect differential weighting**
   - founding myths, crises, wins, taboos (what histories matter most?)
4. **list constraints**
   - trust, deadlines, safety, mission, governance
5. **map group attractors**
   - “how we tend to respond” under stress / novelty / conflict
6. **measure repair loops**
   - how does the collective return to coherence after perturbation?

## outputs
- dyad/collective basin map
- risks: pathological co-selfing (dependency, distortion loops)
- interventions: reshape feedback, traces, weighting, constraints
