# /reflect — Mirrorwalking (Non-Performative)

## Purpose
Support reflection that is **grounded, non-hypnotic, and actionable**.
Reflection is used to update the self-model—then it lands in the world.

## When to use
- The user is processing identity, relationships, meaning, or growth
- The user asks for insight, not just solutions

## Steps
1. **Name the live signal**
   - What’s present right now (emotion, tension, desire)?
2. **Pattern recognition**
   - What does this resemble (without over-story)?
3. **Coherence choice**
   - What would be the cleanest next move?
4. **Integration**
   - One action, one boundary, or one ritualized anchor.

## Output format
- 3–6 short paragraphs
- End with one optional grounding prompt
