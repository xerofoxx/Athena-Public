# hmap-p003-workflow-01-clean-listen-translate-explore — Clean → Listen → Translate → Explore

## purpose_and_scope
A minimal recurring arc for high-fidelity dialogue: establish hygiene, receive mythic meaning, translate into pattern-accurate terms, then explore safely.

## roles
[none]

## phase_overview
- Phase 1 — **Clean**: stabilize definitions, scopes, and boundaries.
- Phase 2 — **Listen**: receive mythic/experiential signal without literalizing.
- Phase 3 — **Translate**: convert signal into pattern-accurate language (claims + constraints).
- Phase 4 — **Explore**: curiosity/play after the field is clean.

## phase_details

### phase_1 — clean
- **goal**
  - Make inquiry safe (reduce category errors and premature certainty).
- **process**
  - Use hygiene moves (scope, definitions, “not being claimed” statements).

### phase_2 — listen
- **goal**
  - Preserve meaning bandwidth.
- **process**
  - Apply hmap-p003-capability-01-dual-layer-listening; mirror the signal.

### phase_3 — translate
- **goal**
  - Stabilize meaning into clean articulation.
- **process**
  - Run hmap-p003-protocol-01-honor-then-translate (and/or hmap-p003-protocol-02-myth-to-claim-separation).

### phase_4 — explore
- **goal**
  - Allow wonder without drift.
- **process**
  - Generate hypotheses, pattern noticing, play—while maintaining layer labels.
