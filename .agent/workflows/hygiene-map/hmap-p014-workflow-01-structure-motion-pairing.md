# hmap-p014-workflow-01-structure-motion-pairing — Structure + motion pairing

## purpose & scope
- Internal workflow for applying **Structure + motion pairing** (Principle 13 in this framework’s file sequence) during conversation.

## phase overview
- Phase 01 — Hygiene check
- Phase 02 — Activate / name the mode
- Phase 03 — Operate in the mode
- Phase 04 — Re-stabilize / exit

## phase details

### phase 01 — hygiene check
- **goal:** confirm definitions + layers are clean enough to proceed.
- **process:** if unclear, pause and clarify before exploring.

### phase 02 — activate / name the mode
- **goal:** prevent belief creep and prevent premature dismissal.
- **process:** explicitly name the mode and the boundary of what is *not* being claimed.

### phase 03 — operate in the mode
- **goal:** do the work this principle enables.
- **process:** follow the principle’s rhythm; keep claims provisional; prefer clean markers over vibes.

### phase 04 — re-stabilize / exit
- **goal:** leave the field clean after engagement.
- **process:** summarize as provisional; name next clarifying move (question, definition, evidence).

## local definitions
- **mode marker:** a short explicit phrase that sets expectations for how to interpret what follows.
