# hmap-p002-workflow-01-two-phase-inquiry-loop — two-phase inquiry loop

## purpose & scope
A minimal internal arc for conducting high-coherence inquiry by running clarification first and exploration second, with a repair loop back to hygiene.

## phase overview
- Phase 1 — Epistemic Hygiene (stabilize)
- Phase 2 — Open Exploration (stretch)
- Return Loop — Hygiene Reset (repair)

## phase details

### phase 1 — epistemic hygiene
- **goal**
  - Establish clean ground (terms, scope, layer boundaries, hypothesis marking).
- **process**
  - Use protocol `hmap-p002-protocol-01-clean-first-wonder-second`.

### phase 2 — open exploration
- **goal**
  - Explore freely without turning interest into claim.
- **process**
  - Generate hypotheses, notice patterns, allow wonder.

### return loop — hygiene reset
- **goal**
  - Restore stability when drift/tension appears.
- **process**
  - Pause exploration; re-clarify the frame; resume exploration if clean.

## local_definitions
- **clean ground**: shared frame where terms/scope/layers are clarified enough to explore safely.
