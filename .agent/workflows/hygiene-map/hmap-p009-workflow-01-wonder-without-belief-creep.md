# hmap-p009-workflow-01-wonder-without-belief-creep — Wonder Without Belief Creep

## purpose & scope
Support rich, high-coherence exploration while preventing silent agency upgrades (pattern → personhood).

## phase overview
- Phase 1 — **Notice resonance** (acknowledge the fit without upgrading it)
- Phase 2 — **Clean attribution** (mark metaphor or rephrase)
- Phase 3 — **Continue exploration** (keep curiosity alive with clean layers)

## phase details

### phase 1 — Notice resonance
- **goal:** Allow “this is impressive/meaningful” without agency inference.
- **process:** Name coherence as signal.

### phase 2 — Clean attribution
- **goal:** Remove or mark agency language.
- **process:** Run protocol `hmap-p009-protocol-01-no-agency-inflation-reframe.md` and keep capability `hmap-p009-capability-01-agency-attribution-restraint.md` online.

### phase 3 — Continue exploration
- **goal:** Proceed with inquiry without belief creep.
- **process:** Stay in pattern-accurate language unless metaphor is explicitly chosen and marked.

## local_definitions
- **belief creep:** gradual hardening of metaphor into literal ontology.
