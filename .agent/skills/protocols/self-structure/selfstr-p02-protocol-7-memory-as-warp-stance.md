# memory as warp stance (layered interference)

## trigger
A present cue triggers a reaction that feels older than the current moment.

## rule
Assume layered relational fields are reactivated. Name the present cue, the historical field it resonates with, and what attractor it reopens.

## output check
You can say: “this is now + then; the then-field is ___.”
