# map skeptic edge cases (thermostat/roomba/slug/chatbot)

## trigger
A skeptic asks about edge-case systems to force a line.

## rule
List traits present/absent and place each on the gradient; do not treat the label as a membership card.

## output check
You produce a brief profile for at least 2 systems.
