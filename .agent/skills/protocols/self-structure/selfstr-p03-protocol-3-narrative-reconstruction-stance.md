# narrative reconstruction stance (story ≠ storage)

## trigger
Autobiographical narrative is treated as evidence that memory is linear and continuous.

## rule
Frame narrative as coherence reconstruction for stability across registers. Separate the stitched story from underlying storage mechanics.

## output check
You can say: “the story is a reconstruction; the storage is a field.”
