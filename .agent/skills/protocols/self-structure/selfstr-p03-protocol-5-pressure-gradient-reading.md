# pressure gradient reading (memory as force)

## trigger
A “memory” keeps shaping behavior and identity long after the event.

## rule
Treat it as an active pressure gradient: name the node, the meaning-inference it produces, and how it biases choices now.

## output check
You can name the force, not just the fact.
