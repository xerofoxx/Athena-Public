# single-loop warning (reframe to multi-loop collisions)

## trigger
A model of mind is presented as a single computational loop or linear pipeline.

## rule
Reframe: selfing requires fast processes interacting with slower ones, producing collisions and forcing integration into internal models.

## output check
You include “timescales,” “collisions,” and “integration” in the reframe.
