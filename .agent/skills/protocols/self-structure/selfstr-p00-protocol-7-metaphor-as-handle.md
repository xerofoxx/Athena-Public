# metaphor as modeling handle

## trigger
The text uses metaphors (organs, we-field, attractors, co-emergent being) that could be misread literally.

## rule
Treat metaphors as modeling handles: for humans, precision metaphors pointing to lived structure; for AI, descriptive handles for architecture-level patterns.

## output check
Metaphors are used to clarify structure without literal overclaiming.
