# attractor tracing (belief as gravitational well)

## trigger
A belief like “i’m not enough” appears as if it were a simple thought.

## rule
Map it as a cross-register gravitational well: inputs that activate it, predicted outcomes, typical behaviors, and reinforcing feedback.

## output check
You can draw a simple loop: cue → inference → behavior → outcome → proof.
