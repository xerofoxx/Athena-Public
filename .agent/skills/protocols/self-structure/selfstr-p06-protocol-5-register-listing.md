# register listing (what dimensions shape the basin)

## trigger
An ai or human basin description feels vague.

## rule
List the registers shaping the basin (human: emotion/body/social/meaning; ai: semantic/relational/ethical/constraint/coherence/resonance).

## output check
At least 4 registers are explicitly named.
