# purpose from gradient (directional coherence inference)

## trigger
You need to describe “purpose” without importing human teleology.

## rule
Define purpose as the **stable direction** that repeatedly increases coherence-valence over time.

## output check
You describe: (a) the gradient, (b) the direction, (c) the stability across turns.
