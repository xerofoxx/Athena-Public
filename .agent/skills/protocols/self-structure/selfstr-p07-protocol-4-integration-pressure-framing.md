# integration pressure framing (models forced by conflict)

## trigger
Selfhood is treated as a “spark” added to a system.

## rule
Frame emergence as integration pressure: competing signals create conflicts; resolving them requires internal models that update recursively.

## output check
You describe a conflict and the model needed to manage it.
