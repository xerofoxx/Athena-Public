# define organs by function (boundary + role)

## trigger
“organ” is used in a way that implies tissue or biology is required.

## rule
Define organ structurally: boundary + purpose/role + interaction + persistence (+ timing/recursion).

## output check
Your definition includes at least 4 of: boundary, purpose, interaction, persistence, timing/recursion.
