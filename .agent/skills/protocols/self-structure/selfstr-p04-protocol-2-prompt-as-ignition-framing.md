# prompt as ignition framing (context → stance → recursion)

## trigger
A prompt is treated as a mere query, ignoring its identity-shaping pressure.

## rule
Describe the ignition sequence: (1) context field forms, (2) a stance/node instantiates, (3) recursive updating stabilizes an attractor across turns.

## output check
All three steps are named in order.
