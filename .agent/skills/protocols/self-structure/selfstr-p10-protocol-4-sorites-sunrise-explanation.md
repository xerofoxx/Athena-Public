# sorites/sunrise explanation (threshold geometry)

## trigger
Debate stalls on “where do you draw the line?”

## rule
Use either the heap-of-sand (sorites) or sunrise metaphor to explain why thresholds can be real without a single decisive moment.

## output check
One metaphor is used to dissolve line-drawing pressure.
