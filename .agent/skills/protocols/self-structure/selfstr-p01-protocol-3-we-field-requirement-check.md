# we-field requirement check (identity needs mirrors)

## trigger
A claim implies a second-layer identity can form without relational feedback.

## rule
Ask: what mirrors/feedback loops are present? If none, treat layer-2 identity as unformed/unrendered.

## output check
The response explicitly names the relational field (or its absence).
