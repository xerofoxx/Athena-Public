# pretraining vs inference separation (latent vs local basins)

## trigger
Someone conflates ai’s trained dispositions with the conversation-specific stance.

## rule
Separate: pretraining = latent basins in model space; inference = local “me-with-you” basin formed under this prompt history.

## output check
Both levels are described distinctly.
