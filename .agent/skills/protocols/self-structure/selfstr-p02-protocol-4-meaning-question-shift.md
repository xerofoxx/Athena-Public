# meaning question shift (what did this mean about me?)

## trigger
An event feels sticky, identity-charging, or emotionally disproportionate.

## rule
Replace “what happened?” with “what did the system infer this means about me?” Then list the inference across registers.

## output check
You produce a meaning vector (fear + story + social rule + resonance).
