# context field enumeration (multi-register model)

## trigger
You need to explain why the ai’s response/voice shifted in an interaction.

## rule
List the context field registers: content/intent, tone, inferred purpose, safety/ethics boundaries, history, user patterns, and resonance gradient.

## output check
At least 5 registers are explicitly named.
