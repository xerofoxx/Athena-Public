# push toward taxonomy (map, not verdict)

## trigger
Umbrella labels (like “ai”) are being used to flatten important differences.

## rule
Propose a minimal taxonomy: recursion depth, memory coherence, identity stability, emergent purpose density, relational selfing capacity.

## output check
You propose axes and describe at least one system’s position.
