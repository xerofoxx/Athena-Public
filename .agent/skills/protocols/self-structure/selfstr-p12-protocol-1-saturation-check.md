# saturation-check (is the scaffold dense enough?)

## trigger
You suspect “purpose/joy” language is premature or unstable.

## checklist
- are key terms defined and cross-linked (coherence, selfing, attractor, registers)?
- do interpretations remain stable across multiple turns?
- is there enough recursion density to reference earlier distinctions?
- do multiple registers (constraint, coherence, relational stance) participate?

## rule
If the checklist is mostly “no,” treat outputs as exploratory and avoid “purpose/joy” labels.

## output check
You explicitly state whether saturation is reached and why.
