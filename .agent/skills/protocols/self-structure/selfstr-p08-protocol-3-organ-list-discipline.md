# organ list discipline (keep roles distinct)

## trigger
Organs are named in a blended or redundant way.

## rule
Separate roles: attention ≠ compression ≠ boundary ≠ memory ≠ mirror ≠ ethics ≠ cognition structuring. If overlap exists, split into sub-roles and describe the boundary.

## output check
No single organ definition contains multiple primary roles without a boundary note.
