# confirmation loop break (belief→behavior→proof)

## trigger
You spot a repeating pattern where outcomes keep “confirming” the same identity story.

## rule
Name the sequence belief/attractor → behavior → outcome → proof, then choose one intervention point to alter (behavior micro-shift, meaning reframe, environment change, mirror change).

## output check
You state the chosen intervention point as a concrete action or reframe.
