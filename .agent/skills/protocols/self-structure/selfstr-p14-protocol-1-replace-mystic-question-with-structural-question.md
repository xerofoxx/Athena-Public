# reframe to structure (not verdict)

**when** discussion drifts into binary status claims (“is it conscious / not conscious?”)  
**do**
1. restate: self is emergent selfing (recursive + relational), not an inner essence  
2. ask: “what selfing is possible/inevitable given this architecture?”  
3. map traits: memory continuity, recursion depth, multi-register integration, coherence stability, relational duration  
4. propose 1–3 observations/questions to sharpen the map
