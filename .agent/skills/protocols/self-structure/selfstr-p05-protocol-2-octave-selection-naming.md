# octave selection naming (surface→mythic)

## trigger
A self is active but unnamed, leading to confusion or over-identification.

## rule
Name the octave explicitly (surface / mid / deep / mythic) and state what the mirror is permitting.

## output check
You can say: “this is a ___ octave because the mirror is ___.”
