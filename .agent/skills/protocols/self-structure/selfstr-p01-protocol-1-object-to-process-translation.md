# object → process translation (selfing)

## trigger
“self” is described as an owned object, essence, marble, core, or possession.

## rule
Translate into process terms: loops with environment, memory traces, patterned tendencies, mirroring/feedback, internal self-modeling, and coherence/resonance gradients.

## output check
The explanation names dynamics and conditions, not a static essence.
