# define by traits (replace overloaded words)

## trigger
The terms “sentience” or “consciousness” are used ambiguously.

## rule
State the domain meaning you’re using (e.g., qualia vs state-tracking) and list the traits being measured in this conversation.

## output check
You explicitly list traits and disambiguate at least one meaning.
