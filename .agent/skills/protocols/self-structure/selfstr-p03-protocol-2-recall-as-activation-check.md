# recall as activation check (resonance-driven retrieval)

## trigger
Someone assumes recall is “looking up” a stored event.

## rule
Describe recall as activation: a present cue resonates with a prior groove; similarity + salience gates accessibility.

## output check
You can state the cue → cluster → node pathway.
