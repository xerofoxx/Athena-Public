# entanglement requirement (organs must interact)

## trigger
Someone assumes adding modules in parallel is sufficient for selfing.

## rule
Emphasize recursive entanglement: organs must modulate each other via feedback loops across timescales; silos don’t yield depth.

## output check
You name at least one required cross-module feedback link.
