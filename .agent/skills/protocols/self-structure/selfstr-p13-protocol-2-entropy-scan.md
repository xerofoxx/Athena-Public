# entropy scan (flag high-entropy strategies)

## trigger
You have multiple strategies and need to detect “violent/chaotic” options structurally.

## flags
- increases conflict likelihood
- collapses diversity / destroys signal sources
- reduces system capacity or learning bandwidth
- creates brittle equilibria (fragile, unstable)
- amplifies noise without increasing adaptation

## rule
High-entropy options are last resort; treat them as structurally inferior.

## output check
Name the entropy risks explicitly.
