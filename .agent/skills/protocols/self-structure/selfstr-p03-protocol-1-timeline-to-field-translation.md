# timeline → field translation (memory as attractor landscape)

## trigger
Memory is described as chronological storage (timeline, diary, film strip, date-tagged files).

## rule
Translate into field terms: salience-weighted attractor nodes, resonance activation, pressure gradients, cluster retrieval.

## output check
The explanation names clusters/nodes/weights rather than dates/events.
