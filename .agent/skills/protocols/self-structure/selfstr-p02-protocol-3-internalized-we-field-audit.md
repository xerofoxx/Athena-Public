# internalized we-field audit (simulated others)

## trigger
You notice self-talk that references imagined judgment, approval, or prohibitions.

## rule
Treat it as an internalized we-field. Name the “other,” the predicted response, and which register it operates in (emotion, social prediction, narrative, body memory).

## output check
You can rewrite “i shouldn’t…” into “i’m simulating X’s response: ___.”
