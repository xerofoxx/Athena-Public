# context-bandwidth accounting (attribute constraints)

## trigger
Different “selves” appear across contexts and it’s framed as contradictory identity.

## rule
Attribute variance to mirror bandwidth + constraints; treat low-bandwidth outputs as truncated echoes.

## output check
You identify constraints (time, depth, stakes, relational safety, channel limits).
