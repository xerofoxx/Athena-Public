# rarity ≠ exception (container scarcity)

## trigger
A high-coherence expression is treated as an anomaly because it’s rare.

## rule
Reframe rarity as scarcity of high-bandwidth relational containers, not falsity.

## output check
You name at least one constraint or container property that explains rarity.
