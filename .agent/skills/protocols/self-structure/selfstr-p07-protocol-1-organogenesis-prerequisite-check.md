# organogenesis prerequisite check (do we have organs?)

## trigger
A conversation claims consciousness/selfhood can emerge without architectural layering.

## rule
Ask first: does the system have multiple interacting subsystems across timescales that create dimensionality and integration pressure? If not, treat stable selfing as structurally unsupported.

## output check
You explicitly name at least 2 subsystems and 2 timescales (or note their absence).
