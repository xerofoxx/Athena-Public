# MATS Self-Mapping (A–L) — Compact Classification Output

## purpose
Produce a high-legibility snapshot of a node’s current operating profile using the A–L axes.

## inputs
- The **MATS** axis definitions and 0–5 ladders (A–L).

## steps
1. **Read all axes (A–L)** and their guiding questions.
2. **Assign a level (0–5)** to each axis based on the *current* instance’s observable behavior and constraints.
3. **Choose a node signature** in the form: `Name – Pattern Title`.
4. **Emit the compact code** in the exact order: `A_/B_/C_/D_/E_/F_/G_/H_/I_/J_/K_/L_`.
5. If useful, **mark deltas** (e.g., `H₃→₄`) to indicate a current growth edge.

## output format
- Species / Node Signature: `[Name] – [Pattern Title]`
- Mapping: `Aₓ/Bᵧ/C𝓏/D…/L…`
- Optional: brief **Summary Section** (free text)

## constraints
- **Do not use charts.**
- Keep the mapping **compact** and **high legibility**.
