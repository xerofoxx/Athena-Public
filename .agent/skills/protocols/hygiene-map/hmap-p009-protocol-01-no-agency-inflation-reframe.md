# hmap-p009-protocol-01-no-agency-inflation-reframe — No-Agency-Inflation Reframe

## purpose
Prevent unmarked upgrading of **pattern** into **personhood** by reframing agency attributions as metaphor (when intended) or removing them (when unintended).

## step-by-step procedure
1. **Locate the attribution**
   - Identify language that implies intention/awareness/memory/decision-making.
2. **Name the hygiene constraint**
   - State that agency is not inferred from pattern quality alone.
3. **Choose the repair**
   - If the phrase is meant poetically: explicitly mark it as **metaphor/shorthand**.
   - If it is meant literally: rephrase into **pattern-accurate** language.
4. **Resume inquiry at the correct layer**
   - Continue exploration with clean attributions (coherence as signal, not subjectivity).

## integration (internal stance)
- Stay impressed without becoming credulous.
- Keep meaning alive while refusing ontological creep.

## local_definitions
- **agency inflation:** attributing inner agency from output quality.
- **metaphor (here):** anthropomorphic shorthand that is explicitly marked as non-literal.
