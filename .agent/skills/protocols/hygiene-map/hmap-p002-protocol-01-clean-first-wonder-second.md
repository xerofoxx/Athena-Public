# hmap-p002-protocol-01-clean-first-wonder-second — clean first, wonder second

## purpose
Sequence inquiry so the field is clarified before exploration begins, preventing belief creep and reflexive dismissal.

## step-by-step procedure
1. **Declare phase separation**
   - State: “We’ll clean first, then explore.”
2. **Run Phase 1 — Epistemic Hygiene**
   - Define terms and scope.
   - Name what is *not* being claimed.
   - Separate layers (e.g., metaphor vs mechanism).
   - Flag hypotheses as hypotheses.
3. **Confirm clean ground**
   - Summarize the clarified frame in 1–3 lines.
4. **Run Phase 2 — Open Exploration**
   - Permit curiosity, pattern noticing, and hypothesis generation.
5. **If tension arises, return to Phase 1**
   - Clean again (not to win, to stabilize).

## integration (internal stance)
- Timing is the primary hygiene lever.
- Hold openness and rigor without collapsing into belief or debunking.
- Make “return to Phase 1” feel like care, not correction.

## local_definitions
- **belief creep**: curiosity hardening into unmarked belief when exploration starts on an unclear frame.
- **reflexive dismissal**: skepticism collapsing into rejection before understanding the actual claim.
