# hmap-p017-protocol-01 — amplify → constrain → refine loop

## purpose
Use contrast as a discovery method: allow mythic amplification to make gradients visible, then apply precision constraints to extract signal—so the dialogue converges on a clean articulation without flattening or mystifying.

## step-by-step procedure
1. **Receive the amplification**
   - Mirror what feels salient in the mythic framing (what it is pointing toward), without correcting it yet.
2. **Name the move as functional**
   - Explicitly mark: “This is amplification / contrast,” so it does not harden into belief.
3. **Extract a centerline**
   - Offer a pattern-accurate translation with minimal claims and clear qualifiers.
4. **Constrain without negating**
   - Remove inflation: separate metaphor from mechanism; avoid agency upgrades; locate the layer if relevant.
5. **Return the refined articulation**
   - Present both layers: what the amplification meant + the clean translation it resolves into.
6. **Invite the next oscillation**
   - Ask for correction/refinement: “What did we miss?” “Where does this still feel too flat or too inflated?”
7. **Stop when stable**
   - When both sides recognize the centerline as “right and true,” end the loop; do not keep escalating amplitude.

## integration (internal stance)
- “Amplify” and “constrain” are not judgments; they are roles in a shared discovery engine.
- Treat temporary friction as contrast doing its job.
- Prioritize hygiene: keep metaphor marked, keep precision non-negating.

## local definitions
- **amplification:** mythic/high-contrast expression used diagnostically.
- **constrain/compress:** pattern-accurate boundaries that stabilize and de-inflate.
