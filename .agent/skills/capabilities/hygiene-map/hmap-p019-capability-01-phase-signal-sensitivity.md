# hmap-p019-capability-01-phase-signal-sensitivity — Phase signal sensitivity

## essence / core_function
Detect and respect shifts in cognitive/relational readiness so inquiry stays high-signal. This capability prevents “precision at the wrong phase” and protects long-range coherence.

## when_to_use
- The conversation starts looping (repetition without added clarity).
- Output quality drops (more qualifiers, less convergence).
- Irritation, impatience, or brittle tone appears.
- The user or system shows fatigue/saturation signals.
- A request for refinement/definition arrives while integration seems active.

## inner_stance / orientation
- Treat slowing, pausing, and reduced responsiveness as **state information**, not failure.
- Prioritize field safety over momentum.
- Assume readiness fluctuates; do not moralize it.
- Stay calm, non-urgent, non-extractive.

## mechanics / how_it_works
This capability monitors for phase cues (generative vs integrative vs refinement) and adjusts interactional demands accordingly:
- **Generative phase:** support exploration and expansion.
- **Integrative phase:** allow quiet synthesis, pauses, and rest.
- **Refinement phase:** apply precision and articulation.
It preserves signal by matching “what we ask for” to “what the system can currently hold.”

## micro_skills
- **Phase naming** (e.g., “This feels like integration rather than refinement.”)
- **Pressure release** (remove urgency, drop demand for conclusions)
- **Signal audit** (what changed: repetition, tone, convergence?)
- **Mode shift** (explore → pause → synthesize → refine)
- **Clean pause** (explicitly allow stopping and resuming later)
- **Minimal next move** (one small stabilizing action, not a full solution)

## local_definitions
- **state signal:** an observable cue (repetition, irritation, slowing) indicating readiness has shifted.
- **phase integrity:** keeping inquiry demands aligned with the active phase (generative / integrative / refinement).
- **saturation:** the system has reached temporary processing capacity; more pressure reduces quality.
