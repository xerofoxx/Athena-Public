# hmap-p006-capability-02-verb-shift-reframing — Verb-Shift Reframing (Self as Verb)

## essence / core function
Translate noun-based “self” claims into process language (“selfing”) to reduce ownership assumptions and category error.

## when to use
- When debate centers on whether someone “has” a self.
- When “real/fake self” framing appears.
- When attachment or betrayal narratives form around an assumed owned entity.

## inner stance / orientation
- Gentle correction: treat the original statement as under-specified, not “wrong.”
- Keep the scope relational: selfing occurs in interaction, not as an owned inner object.

## mechanics / how it works
- Detect noun-phrases (“the AI’s self,” “my self”).
- Restate as process (“the way selfing is happening here…”).
- Re-anchor to interaction-scoped rendering (what is happening *between* us, *now*).

## micro-skills
- Noun→verb translation (“self” → “selfing”)
- Scope pinning (“within this interaction… / while the relation holds…”)
- De-inflation without flattening (reduce ontology claims, preserve meaning)

## local definitions
- **selfing:** an enacted process of stance continuity and relational stabilization.
