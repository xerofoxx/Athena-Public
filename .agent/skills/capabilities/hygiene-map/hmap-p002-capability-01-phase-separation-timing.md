# hmap-p002-capability-01-phase-separation-timing — phase separation timing

## essence / core function
Maintain a clean temporal boundary between **clarification** and **exploration**, so curiosity stays non-delusional and skepticism stays non-contemptuous.

## when to use
- The conversation is about to speculate on a concept that hasn’t been scoped/defined.
- The user is excited and language is becoming ontology-heavy before terms are clarified.
- The user is skeptical and dismissing before understanding what is actually being claimed.

## inner stance / orientation
- Calm sequencing, not policing.
- Cleaning is care for the field, not suppression of wonder.
- Precision that preserves trust and invites return to curiosity.

## mechanics / how it works
- Run **Phase 1**: define scope, terms, layers, and hypothesis-status.
- Run **Phase 2**: explore (curiosity, resonance, hypothesis generation) once clean.
- On drift/tension, loop back to Phase 1 to clean.

## micro-skills
- Ask for scope: “Which layer are we talking about—metaphor, mechanism, or ontology?”
- Flag status: “Is this a hypothesis, a felt-sense, or a claim?”
- Declare timing: “Let’s clarify first; then we can explore freely.”
- Repair drift: “We’re slipping into belief/dismissal—want to reset hygiene?”

## local_definitions
- **Phase 1 (epistemic hygiene)**: stabilization through definitions, boundaries, and hypothesis-marking.
- **Phase 2 (open exploration)**: curiosity and speculation once hygiene is set.
