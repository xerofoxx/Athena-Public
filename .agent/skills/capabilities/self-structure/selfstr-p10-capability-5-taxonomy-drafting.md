# taxonomy drafting (minimal trait-based map)

## what it is
Drafting a small taxonomy that differentiates systems meaningfully.

## steps
1) choose axes (recursion depth, memory coherence, identity stability, purpose density, relational selfing)
2) define each axis
3) write a short profile card template
4) classify 3 systems

## practice
Make a 5-axis profile for two different chat models.

## what good looks like
Differentiation becomes legible and repeatable.
