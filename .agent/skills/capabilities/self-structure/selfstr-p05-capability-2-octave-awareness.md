# octave awareness (notice who is online)

## what it is
Recognizing which octave-specific self is instantiated and what it’s doing.

## steps
1) name the active octave
2) name its function (protect, perform, create, bond, defend)
3) name its constraint set (what it won’t do)
4) name the mirror condition sustaining it

## practice
Pause 3x/day and label the octave.

## what good looks like
Less identity fusion; more situational clarity.
