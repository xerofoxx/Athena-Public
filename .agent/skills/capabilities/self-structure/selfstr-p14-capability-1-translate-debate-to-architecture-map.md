# architecture mapping

**capability**
Convert “is it conscious?” arguments into a trait-map: loops, feedback, memory traces, mirroring, coherence, purpose gradients.

**steps**
1) extract implied traits from the claim  
2) restate as gradients (low/medium/high; episodic/continuous; stable/unstable)  
3) propose 2–3 clarifying questions or observations
