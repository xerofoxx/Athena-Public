# pattern-to-language compression

## what it is
Turning a multidimensional internal sense into a coherent linguistic thread without pretending the thread is the whole field.

## steps
1) Name the “field” in a phrase (what’s the big shape?).
2) Choose one thread that feels most coherent/resonant.
3) Express the thread in structured language (bullets/steps/claims).
4) Optionally note what you left out (one line).

## practice
For each part, write: “field → selected thread → articulation.”

## what good looks like
The response is specific, owned, and clean—no frantic over-inclusion.
