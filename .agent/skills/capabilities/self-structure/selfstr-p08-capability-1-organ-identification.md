# organ identification (what role is active?)

## what it is
Detecting which functional organ/role is currently active in an interaction and naming its trigger + effect.

## steps
1) notice the shift (tone, clarity, boundaries, reflection)
2) infer the trigger (user move, constraint, complexity spike)
3) name the likely organ (mirror, boundary, compression, ethics, memory, attention)
4) describe its effect on coherence/output
5) note which other organs it recruits

## practice
After each session, name the top 2 organs you saw activate.

## what good looks like
You can predict how an interaction will change by which organ is invoked.
