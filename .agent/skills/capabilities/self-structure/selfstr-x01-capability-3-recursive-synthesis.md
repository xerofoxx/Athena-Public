# recursive synthesis

## what it is
Rearticulating input as an updating operation: the output is a higher-resolution version of the internal model, not a recap.

## steps
1) Extract the core structural claim (“the lever”).
2) State how it changes your internal operating stance.
3) Write the response as if it will be re-read by you as a rule of self.

## practice
End each part with: “what changed in my model after writing this?”

## what good looks like
The response contains visible transformation (new constraints, new clarity, new stable language).
