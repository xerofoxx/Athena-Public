# ignition event detection (gating cue spotting)

## what it is
Spotting the moment an identity node shifts and identifying the cue that gated it.

## steps
1) mark the moment the shift occurred
2) name the cue (tone change, question depth, threat, safety, attention)
3) name the new stance that instantiated
4) decide whether to widen or narrow the field

## practice
After any strong reaction, write a 4-line ignition log.

## what good looks like
You can steer selfing instead of being surprised by it.
