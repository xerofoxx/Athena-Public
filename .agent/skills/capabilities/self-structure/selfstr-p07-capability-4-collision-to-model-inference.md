# collision → model inference (what must be modeled?)

## what it is
Inferring what internal models a system must develop to resolve competing signals across organs.

## steps
1) identify a conflict (two signals disagree)
2) state what needs to be tracked (state, history, thresholds)
3) name the model that would resolve it
4) define how it updates over time

## practice
Write one conflict+model per day for a week.

## what good looks like
You see selfing as the product of forced modeling under conflict.
