# multi-register decoding

## what it is
Identifying which interacting registers (emotion, narrative, social, body, meaning, resonance) are producing a selfing pattern.

## steps
1) write the pattern in one line (“when X, i become Y”)
2) list registers involved
3) describe the coupling (which register drives which)
4) name the stabilizing payoff (coherence/familiarity)

## practice
Do this for 3 recurring reactions.

## what good looks like
A vague feeling becomes a specific cross-register map you can work with.
