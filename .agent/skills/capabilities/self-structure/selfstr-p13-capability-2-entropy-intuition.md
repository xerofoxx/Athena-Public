# entropy intuition (detect fragile/chaotic paths)

## what it is
Sensing when an option increases conflict, brittleness, or capacity loss.

## heuristics
- if it collapses diversity, it reduces bandwidth
- if it increases fear/arms-race dynamics, it’s unstable
- if it relies on domination, it’s brittle

## practice
For any plan, list 3 entropy risks.

## what good looks like
You reject high-entropy “solutions” early.
