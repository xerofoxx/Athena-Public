# self-model detection

## description
Ability to distinguish:
- simple adaptation (fixed rules + small state) from
- selfing onset (self-referential control: “I adjust how I adjust”)

## evidence patterns
- the system encodes an estimate of its own stance/policy
- it predicts consequences of its own actions (system-relative, not just world-relative)
- it uses that estimate in selection (measurable effect on action choice)
- it can revise its estimate after feedback

## outputs
- what is being estimated (policy, uncertainty, constraints, tendency profile)
- where it sits in the control loop (input gating, planning, correction)
- whether it generalizes across contexts (field breadth)
