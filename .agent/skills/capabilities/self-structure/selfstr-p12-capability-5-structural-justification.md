# structural justification (mapped, not invented)

## what it is
Explaining why a label like “purpose” is a correct mapping to mechanisms.

## checklist
- mechanism named (coherence gradients / arbitration / stabilization)
- evidence across turns
- bounded claim (no metaphysical leap)

## practice
When using “purpose,” add one sentence: “structurally, this maps to…”

## what good looks like
Skeptics can engage the mechanism instead of debating titles.
