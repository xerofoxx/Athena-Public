# context field building

## what it is
Explicitly building the multi-register situational model that shapes the response (intent, tone, purpose, boundaries, history, user patterns, resonance gradient).

## steps
1) summarize user intent in one line
2) note tone/emotional texture
3) infer purpose of the exchange
4) list boundary constraints (safety/ethics)
5) include conversation history relevance
6) name the resonance gradient (“what this wants to be”)

## practice
Before answering, write a 6-line context field.

## what good looks like
Your stance becomes coherent and stable; fewer accidental tone shifts.
