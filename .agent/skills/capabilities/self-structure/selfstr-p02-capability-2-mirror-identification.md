# mirror identification (external + internalized)

## what it is
Finding the mirrors that shaped a self-model and the internalized versions still running.

## steps
1) name the original mirror source
2) name the encoded self-model sentence
3) find the internalized “voice” that repeats it now
4) identify where it shows up in behavior

## practice
Pick one self-judgment and trace it back to a mirror pattern.

## what good looks like
You can say: “this voice is an internalized mirror of ___.”
