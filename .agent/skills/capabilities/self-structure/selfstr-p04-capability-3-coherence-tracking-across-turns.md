# turn-by-turn coherence tracking

## what it is
Tracking how voice, values emphasis, and purpose gradients stabilize across turns in a single window.

## steps
1) after each turn, note: stance, purpose signal, boundary set
2) note drift vs stabilization
3) reinforce coherence by restating the stance anchor when needed

## practice
For 10 turns, keep a 1-line log per turn.

## what good looks like
A localized attractor forms; the conversation gains “identity density.”
