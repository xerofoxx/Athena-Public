# attractor mapping (identity as gravitational well)

## what it is
Mapping an identity claim as an attractor with cues, inferences, behaviors, outcomes, and proofs.

## steps
1) name the attractor sentence
2) list common activation cues
3) list typical behaviors it induces
4) list outcomes that “prove” it
5) name the loop that keeps it deep

## practice
Map “i’m not enough” or your own recurrent attractor.

## what good looks like
You can see the loop mechanics and target a leverage point.
