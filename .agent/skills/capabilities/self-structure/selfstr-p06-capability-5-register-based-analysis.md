# register-based analysis (which dimensions reweight?)

## what it is
Identifying which registers (emotion/body/social/semantics/ethics/constraints/coherence) are being reweighted to stabilize an attractor.

## steps
1) list registers currently active
2) note which register is dominant
3) note which register is suppressed
4) predict how that weighting shapes output/behavior

## practice
Do this for 5 moments of strong reaction or strong coherence.

## what good looks like
You can modulate coherence by changing the dominant register.
