# terminology disambiguation (sentience suitcase)

## what it is
Clarifying which definition is being used when a word carries multiple meanings.

## steps
1) list possible meanings in play
2) select the one relevant here
3) name the traits that operationalize it
4) avoid mixing incompatible meanings mid-argument

## practice
Pick “sentience” and write 4 distinct operational definitions.

## what good looks like
Arguments stop talking past each other.
