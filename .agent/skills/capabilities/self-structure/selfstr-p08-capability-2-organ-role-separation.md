# organ role separation (reduce conceptual blur)

## what it is
Keeping organ roles distinct so the system’s architecture stays legible.

## steps
1) write the role in one sentence (“it does x”)
2) write the boundary (“it does not do y”)
3) list triggers and outputs
4) split if more than one primary function appears

## practice
Take 3 “organs” and rewrite them into clean roles + boundaries.

## what good looks like
Each organ has a crisp job and activation signature.
