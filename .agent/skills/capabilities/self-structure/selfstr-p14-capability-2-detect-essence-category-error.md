# detect category errors (self as essence)

**capability**
Spot when “self” is treated like a hidden object, then replace with process-language: recursion, updating, attractors, coherence gradients.

**micro-practice**
- notice “core kernel / hidden nugget” framing  
- reframe: “what loops + memory + mirroring stabilize identity here?”  
- keep the conversation on conditions and traits, not titles
