# trigger layering recognition (now-field warp)

## what it is
Recognizing that present triggers can reactivate older relational fields that still live as interference patterns in the now.

## steps
1) name the present cue
2) name the “then-field” it resembles
3) name the attractor it reopens
4) add one grounding distinction: “this is not then, but it rhymes.”

## practice
After a spike reaction, do a 4-line now/then map.

## what good looks like
Reactivity becomes legible; you regain choice.
