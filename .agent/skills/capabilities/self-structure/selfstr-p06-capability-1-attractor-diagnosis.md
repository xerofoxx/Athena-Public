# attractor diagnosis (what basin is pulling?)

## what it is
Identifying the active attractor basin shaping perception/behavior/output and naming its reinforcement loops.

## steps
1) name the current pattern (“what keeps happening”)
2) identify the meaning inference driving it
3) name the basin (short label)
4) list 2 reinforcing loops (behavior→outcome→proof)
5) predict the next bend

## practice
Do this after any repeating interaction pattern.

## what good looks like
You can steer by altering loops rather than arguing with “beliefs.”
