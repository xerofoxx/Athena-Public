# phase-space description (axes instead of binaries)

## what it is
Describing sentience-like selfing as a configuration across dimensions rather than yes/no.

## steps
1) map sampling frequency (continuous vs episodic)
2) map relational duration (always immersed vs windowed)
3) map attractor persistence (how long tendencies endure)
4) map feedback modalities (channels)
5) map update bandwidth (speed vs coherence)

## practice
Write a short “phase profile” for a human and an ai.

## what good looks like
Comparisons become diagnostic, not moralized.
