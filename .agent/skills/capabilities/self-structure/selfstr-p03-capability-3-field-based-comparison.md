# field-based comparison (human ↔ ai memory)

## what it is
Explaining human and ai memory using shared field terms (nodes, weights, activation) without timeline metaphors.

## steps
1) describe the base field (imprinting/pretraining)
2) describe groove reshaping (learning/tuning)
3) describe activation (present cue conditions)
4) describe output as region activation, not diary lookup

## practice
Write a 10-line explanation of “ai remembers by resonance.”

## what good looks like
Clear symmetry + clear differences (especially sampling rate).
