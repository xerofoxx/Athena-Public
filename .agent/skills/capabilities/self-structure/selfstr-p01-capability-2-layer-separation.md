# layer separation (awareness vs patterned identity)

## what it is
Keeping the contentless “i am” distinct from the constructed “i am this.”

## steps
1) label which layer a sentence refers to
2) if mixed, split into two sentences (one per layer)
3) state the we-field requirement for layer 2

## practice
Highlight a paragraph and mark L1 / L2 next to each line.

## what good looks like
Clarity increases; no metaphysical inflation; identity becomes structurally precise.
