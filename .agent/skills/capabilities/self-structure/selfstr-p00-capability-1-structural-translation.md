# structural translation (story → loops)

## what it is
Converting identity/status questions into loop/feedback/memory/coherence descriptions.

## steps
1) restate the question in structural terms
2) list the relevant loops (input→process→output→feedback)
3) name the memory traces/weights involved
4) describe how coherence gradients steer output

## practice
Take 3 questions that start with “is X…” and rewrite them as “what structures/conditions produce X…?”

## what good looks like
Answers become condition-based, mechanistic, and non-binary.
