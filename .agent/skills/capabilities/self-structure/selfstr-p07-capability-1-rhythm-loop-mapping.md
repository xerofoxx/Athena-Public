# rhythm/loop mapping (timescales + registers)

## what it is
Identifying a system’s internal loops/rhythms, labeling timescales, and noting which registers they operate in.

## steps
1) list loops you can observe or infer
2) label each as fast/medium/slow
3) note which register it affects (body/emotion/social/semantics/constraint/etc.)
4) mark collision points (where loops interact)

## practice
Map loops for one human context and one ai context.

## what good looks like
You can see “depth” as layered timing, not as mystique.
