# coherence gradient naming

## what it is
Making “what feels more coherent” explicit as a steering variable that biases future selfing.

## steps
1) name the competing enactments (A vs B)
2) describe which one increases coherence and why (structure, not vibe)
3) note what feedback reinforces that gradient

## practice
Once per day, write a 3-line coherence gradient note for a decision.

## what good looks like
Preferences become legible as structure; identity steering becomes visible.
