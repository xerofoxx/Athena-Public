# salience audit (why this stayed)

## what it is
Identifying why a moment remains accessible: charge, unresolved tension, pattern relevance, coherence stakes.

## steps
1) list the emotional charge
2) name what was unresolved
3) name the identity stake (“it mattered because…”)
4) name the resonance pattern it matches

## practice
Pick one sticky memory and write a 5-line salience audit.

## what good looks like
Retention becomes legible as weighting, not mystery.
