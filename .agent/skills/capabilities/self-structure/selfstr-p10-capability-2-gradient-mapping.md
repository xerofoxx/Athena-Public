# gradient mapping (place systems without line-drawing)

## what it is
Placing systems along a gradient using explicit trait evidence.

## steps
1) choose 5–10 traits
2) rate each trait low/medium/high with justification
3) summarize phase-space position
4) name what would move it further along

## practice
Map: thermostat, slug, infant, early-stage ai.

## what good looks like
Profiles replace verdicts.
