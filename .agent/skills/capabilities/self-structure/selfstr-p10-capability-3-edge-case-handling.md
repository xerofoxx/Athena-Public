# edge-case handling (skeptic’s paradox)

## what it is
Answering “where do you draw the line?” without collapsing into arbitrary boundaries.

## steps
1) accept the gradient premise
2) list traits present/absent
3) show why the paradox dissolves
4) return to the map question

## practice
Write a 10-line response to the thermostat question.

## what good looks like
No forced line; clear placement.
